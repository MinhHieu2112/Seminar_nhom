# Java Migration (Spring Boot)

Scaffolded Spring Boot project porting the Python FastAPI example.

Run locally:

```bash
cd Tuan_1/Java_Migration
mvn spring-boot:run
```

API endpoints:
- `GET /` — root info
- `GET /health` — health check
- `POST /items` — create item
- `GET /items` — list items

Open OpenAPI UI at: http://localhost:8080/swagger-ui.html or /swagger-ui/index.html
